# AI-Driven Customer Segmentation Application

A full-stack application for customer segmentation using AI/ML algorithms with React frontend and Node.js backend.

## Features

- 🤖 AI-powered customer segmentation (RFM Analysis, K-means)
- 💳 Paystack payment integration
- 📊 Real-time analytics dashboard
- 🎯 Targeted marketing campaigns
- 🔐 JWT authentication & authorization
- 📱 Responsive Material-UI design

## Quick Start

### Prerequisites
- Node.js (v16+)
- MongoDB
- Paystack account

### Installation

1. Install backend dependencies:
```bash
cd backend
npm install
```

2. Install frontend dependencies:
```bash
cd frontend
npm install
```

3. Configure environment variables in `.env` files

4. Start MongoDB service

5. Run the application:
```bash
# Terminal 1 - Backend
cd backend
npm run dev

# Terminal 2 - Frontend
cd frontend
npm start
```
